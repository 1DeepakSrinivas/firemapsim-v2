Welcome to the FireMapSim-v2 end-user documentation.

Use this docs site to learn how to:

- set up wildfire simulation projects from scratch,
- run and replay simulations,
- use map and terrain tools effectively,
- work with guided chat or manual setup flows,
- recover from common issues quickly.

## Who this is for

This documentation is for users who operate the FireMapSim-v2 web app:

- analysts exploring wildfire behavior scenarios,
- planners preparing scenario inputs and constraints,
- operators comparing runs and replay outputs,
- collaborators reviewing project states and troubleshooting runs.

## Product flow at a glance

1. Sign in and open `Dashboard`.
2. Create a project and open the project workspace.
3. Set project location and cell grid.
4. Add ignition points/lines and optional fuel breaks.
5. Configure weather and run settings.
6. Start simulation and review replay/metrics.
7. Iterate, compare, and adjust.

This docs site includes two documentation entities:
