## Goal

Configure and run a complete scenario using sidebar and map controls only.

## Steps

1. Open a project workspace.
2. Set `Project Location`.
3. Tune `Cell Grid`.
4. Set weather values or fetch by place/ZIP.
5. Add point/line ignition entries.
6. Add optional fuel breaks.
7. Set simulation timesteps.
8. Start simulation.
9. Review replay and terrain overlays.

## Best practices

- Add at least one ignition before spending time fine-tuning advanced values.
- Keep an initial baseline run, then change one variable at a time.
- Use project rename to keep scenario variants understandable.

## Common blockers

- run button disabled due to missing ignition/location,
- location lock after completed run,
- weather lookup failures.

See [Troubleshooting](/docs/troubleshooting).
