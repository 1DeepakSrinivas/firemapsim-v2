## What this feature does

Account and preference controls support user-level operations without leaving the workspace.

## Available controls

- open account/profile settings,
- view active account identity,
- toggle theme/display preference.

## Where to access

- Dashboard header (theme + user controls),
- Workspace sidebar footer (account + theme controls).

## Troubleshooting

If account actions fail, re-authenticate and reload protected routes.
