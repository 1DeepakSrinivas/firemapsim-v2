## What this feature does

Ignitions define where and how fire starts in the scenario.

## Supported ignition types

- point ignition,
- line ignition.

## Editable ignition properties

- cell coordinates,
- ignition mode/type,
- spread speed,
- team assignment.

## Minimum requirement

At least one ignition segment is required to run simulation.

## Good usage pattern

1. Start with one simple ignition.
2. Run baseline.
3. Add complexity (additional ignitions, mode/speed changes).
