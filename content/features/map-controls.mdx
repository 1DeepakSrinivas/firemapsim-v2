## What this feature does

Map controls provide quick visual and interaction control for scenario setup and review.

## Available controls

- zoom in/out,
- map style switch (`Terrain`, `Street`, `Satellite`),
- interaction HUD guidance for drawing/placement modes.

## Interaction HUD

When entering map interaction modes (pin, line, polygon, etc.), the HUD shows:

- current action instructions,
- confirm option when applicable,
- cancel action button,
- warning hints for invalid placements.

## Best practices

- Use `Terrain` for setup context and layer interpretation.
- Use `Satellite` when aligning scenario to visible land features.
