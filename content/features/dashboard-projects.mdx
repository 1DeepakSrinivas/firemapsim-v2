## What this feature does

The dashboard is the authenticated hub for project lifecycle management.

## Available actions

- create new project,
- search projects by title,
- rename project,
- open project workspace,
- delete project.

## Typical usage

1. Sign in.
2. Open `Dashboard`.
3. Click `New project`.
4. Name/refine project title.
5. Open workspace and configure scenario.

## Notes

- Projects are tied to authenticated user ownership.
- Deleting a project removes associated scenario context.
