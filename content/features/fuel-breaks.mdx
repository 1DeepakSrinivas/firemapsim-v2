---
title: Fuel Breaks
description: Define and manage fuel break/suppression line segments.
---

## What this feature does

Fuel breaks let you represent linear suppression barriers in the simulation area.

## Actions

- add fuel break segment,
- review configured segments,
- remove individual segments.

## Constraints

- project location must be set first,
- validate segment placement against intended boundary.

## Analysis tip

Run a baseline without fuel breaks, then rerun with one break added to isolate impact.
