## What this feature does

Replay exposes time-based simulation output for review and analysis.

## Available controls

- play/pause,
- timeline cursor progression,
- playback-speed aware interpretation,
- live state counts (burning, burned, unburned).

## Output interpretation workflow

1. Start replay from early timeline.
2. Observe spread direction and velocity zones.
3. Cross-check with weather and terrain context.
4. Capture differences between scenario variants.

## Availability

Replay is only available after at least one successful run with saved output.
