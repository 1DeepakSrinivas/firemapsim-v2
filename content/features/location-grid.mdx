## What this feature does

Location and grid settings define simulation extent and spatial resolution.

## Location behavior

- Set project location before ignition/fuel-break setup.
- Changing location before first run is allowed.
- After a successful simulation run, location becomes locked.

## Grid settings

- `Cell resolution`: meters per simulation cell.
- `Cells (side)`: square grid dimension (`N x N`).
- Grid size determines total scenario area represented.

## Practical guidance

- Start with a moderate grid size for initial runs.
- Increase resolution/detail only when needed.
- Keep boundary placement realistic for your scenario objective.
