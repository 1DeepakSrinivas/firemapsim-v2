## What this feature does

Agent chat helps gather setup details and trigger supported workspace actions.

## Supported patterns

- guided setup prompts,
- structured updates to scenario values,
- run simulation trigger,
- replay play/pause commands,
- reset-project trigger.

## Best practices

- keep prompts scenario-specific,
- confirm generated values before final run,
- use chat for orchestration and sidebar controls for precision edits.

## Notes

Manual workflow remains fully supported; chat is optional.
