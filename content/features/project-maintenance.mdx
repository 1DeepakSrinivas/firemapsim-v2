## What this feature does

Project maintenance controls keep project state manageable over repeated runs.

## Available maintenance actions

- rename project,
- relocate project (with reset implications),
- full project reset,
- remove ignitions/fuel breaks selectively.

## Reset behavior

Project reset clears:

- location,
- weather,
- ignitions,
- fuel breaks,
- terrain/session data,
- simulation output,
- chat history tied to reset flow.

Use reset when you intentionally want a clean scenario start.
